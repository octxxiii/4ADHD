    🚀 시작하기

    1. 릴리즈 페이지에서 최신 버전의 anti_adhd.exe 다운로드
    2. 다운로드한 파일 실행
    3. 프로그램을 실행하고 할 일을 관리하세요!

    지원 환경

    현재 Windows 운영체제만 지원합니다.

    🎨 사용법

    • 각 분면의 입력창에 할 일을 입력하고 Enter 키를 누르거나 추가 버튼을 클릭
    • 중요도와 긴급도에 따라 적절한 분면을 선택
    • 체크박스로 완료된 할 일을 표시
    • 더블 클릭으로 할 일의 내용을 수정
    • 우클릭 메뉴로 상세보기, 수정, 삭제 가능
    • ⚙️ 버튼을 클릭하여 설정 창 열기
    • 투명도, 항상 위 등 설정 가능
